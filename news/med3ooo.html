MED3OOO upgrades to Facets 4.61


MED3OOO wanted to deploy TriZetto’s WebSphere-based CWS Web Portal to 5 separate environments for 3 separate clients and perform custom modifications, specific to each customer, to each web portal. The modifications involved Java Server Pages, Javascript, HTML, CSS, MS SQL stored procedures, Web Services, Windows script command files, and WebSphere configuration changes. Some of the modifications made to the CWS web portal were in concert with C# extensions made to the Facets GUI application. All modifications included the creation of supporting documentation and storing all file modifications within the TFS source code repository.
